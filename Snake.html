Line Chatbot for Snake Classification

This project aims to develop a snake species classification system using artificial intelligence and image processing to address the problem of accurately identifying snakebite species, which is crucial for effective treatment. In Thailand, about 69.24% of snakebite victims cannot identify the snake species, leading to treatment errors. The system employs object detection models such as YOLO and various deep learning techniques to detect and classify 10 venomous and non-venomous snake species. It is integrated with a LINE Chatbot, allowing users to send snake images for analysis and receive preliminary information, first aid advice, and emergency contact details. The model achieves high accuracy in detection and classification, enhancing emergency care and decision-making.
